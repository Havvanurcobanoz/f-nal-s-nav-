Bu proje, HTML ve harici CSS kullanılarak geliştirilmiş, mahalle kahvehanesi temasını yansıtan modern ve sade bir web sitesidir.
Hero alanında arka plan görseli ve fade-up animasyonları, bölümlerde kart yapısı, görsel galeride hover efektleri kullanılmıştır.
Responsive tasarımı sayesinde mobil ve masaüstü cihazlarda uyumlu çalışır ve temiz, okunabilir bir kod yapısına sahiptir.
